Projects

Drug Efficacy Analysis - Completed

Project GitHub Repository

  • Full analysis of uploaded autosomal DNA data where individual SNPs in a user's genome is compared to a large database in order to return genetic variants associated with drug response and efficacy.

  • The purpose is to give users an idea of medications that they will respond positively to and to reduce trial and error of trying prescriptions that do not work optimally for them as well as to reduce adverse side effects.

  • Database is built in SQL on its own server and contains 118,000+ genetic associations with various health outcomes. The query can be manipulated to return any present variant's associations with any recorded diseases, mental conditions, physical traits, etc. For the sake of the project's goals, it queries any significant associations between genetic variants and a large list of prescription drugs used to treat mental health conditions and neurodevelopmental disorders.

Cancer Associated Genetic Variations - Completed

Project GitHub Repository

  • This project analyzes a text file of the user's DNA, identifies each individual SNP and compares it to a serverless SQL database consisting of SNPs associated with the development of cancer. After processing, a results page is rendered containing all the genetic variants in the user's DNA associated with cancer risks.

  • Results contain the RSID, Alleles, associated Cancer, and the quantified increase or decrease in risk due to the genetic variant.

  • Built in Python with a Flask framework, static files containing JavaScript, HTML, and CSS, and a database in mySQL.

Gas Price Forecasting - In Progress

Project GitHub Repository

  • A machine learning project testing the accuracy of gas price forecasting among different algorithms.

  • Models used were a Support Vector Regression, a Random Forest Regression, and a Long Short-Term Memory Neural Network.

  • Data comprised of weekly national average gas prices spanning 1993-2023 was acquired through www.eia.gov as an excel spreadsheet and converted to CSV in order to utilize the computing power of AWS Sagemaker for training the SVR and RFR models with the data up to the year 2022.

  • Model accuracy gauged through Mean Absolute Error, Mean Squared Error, Root Mean Squared Error, Mean Absolute Percentage Error, R-Squared values, Symmetric Mean Absolute Percentage Error, as well as Visual Inspection.
